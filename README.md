# xdx.gg LoL Rank Fetcher

A simple Python script that fetches ranked information (SoloQ & Flex) from xdx.gg (EUW).

## Example

THEBAUSFFS  
===== SOLOQ =====  
DIAMOND III - 61 LP  
Wins: 13 | Losses: 15  
Winrate: 46.43%  
===== FLEX =====  
UNRANKED

## Installation

pip install -r requirements.txt

## Run

python xdxggFetcher.py
